# electron-showcase-devtron

> Show how to use [devtron](https://github.com/electron/devtron)


## Dev

```
$ npm install
```

### Test

```
$ npm test
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
